Machine Learning:
Deep learning in computer vision

Over the past few weeks, I focused on using deep learning to create non-linear features to improve the performance of machine learning. I also saw how transfer learning techniques can be applied to use deep features learned with one dataset to get great performance on a different dataset. I built an iPython notebooks for both image retrieval and image classification tasks on real datasets.

In this application, I built a new image retrieval models and explore their results on different parts of my image dataset. Linear regression, classification, training and evaluating a classifier using deep features, are techniques at the core of my capstone project.
